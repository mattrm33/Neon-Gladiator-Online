# Neon Gladiator Online

A production-ready, realtime multiplayer cyberpunk combat game built with Next.js 15, Socket.IO, and Prisma.

## Features
- **Realtime Multiplayer:** Custom Node.js server integrating Socket.IO with Next.js.
- **Matchmaking:** Queue system with ELO placeholders.
- **Economy:** Coins, Shop, Inventory system.
- **Auth:** Stateless secure JWT authentication without external providers.
- **UI:** High-performance Cyberpunk UI with TailwindCSS.
